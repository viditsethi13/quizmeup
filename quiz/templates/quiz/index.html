About Quiz Me Up


Quiz Me Up is a platform that will help you brush your concepts on some essential Computer Science topics. It is an MCQ-based quizzing format. The user will have an option to select a topic of his choice. From the pool of questions on the selected topic, the user will get five random questions. The quiz will be over if the user gives a wrong answer or give all five correct answers. The user can check how many tests they have taken on this platform and also see the details of each test taken by them. In the next version, the user will be able to compare their performance with all other users using this platform. They will also be able to select the number of questions for each quiz.
